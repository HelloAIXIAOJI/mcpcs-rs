# MCP CLI

## What is this?

本项目是基于 modelcontextprotocol/rust-sdk (RMCP) 的 MCP 客户端+服务端


### 客户端

可以手动的调用 MCP Server

### 服务端

提供了一些MCP服务端的示例。
